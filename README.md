# Православные тесты для детей

Веб-приложение для прохождения интерактивных тестов по православной тематике, предназначенное для детей.

## Описание

Приложение предоставляет возможность проходить тесты на различные православные темы, включая библейские события, праздники и притчи. Каждый тест содержит вопросы с вариантами ответов, система автоматически проверяет ответы и выставляет оценку.

## Функциональность

- **Выбор теста**: Главная страница с карточками доступных тестов
- **Настройка количества вопросов**: Возможность выбрать 5, 10, 15 вопросов или все вопросы теста
- **Интерактивное прохождение**: Пошаговое прохождение теста с отображением прогресса
- **Автоматическая проверка**: Мгновенная проверка правильности ответов
- **Система оценки**: Автоматическая оценка результатов с категориями:
  - Отлично (90% и выше)
  - Хорошо (70-89%)
  - Нужно повторить (менее 70%)
- **Просмотр результатов**: Детальный разбор ответов с указанием правильных и неправильных вариантов

## Использование

### Запуск приложения

1. Запустите локальный веб-сервер с помощью скрипта:
   ```bash
   ./start.sh
   ```
   Или используйте Python напрямую:
   ```bash
   python3 -m http.server 8000
   ```

2. Откройте в браузере: `http://localhost:8000`

### Прохождение теста

1. Выберите интересующий тест из списка на главной странице
2. Укажите желаемое количество вопросов (5, 10, 15 или все)
3. Нажмите кнопку "Начать тест"
4. Отвечайте на вопросы по одному, выбирая один из предложенных вариантов
5. После завершения теста просмотрите результаты и детальный разбор ответов

## Формат данных теста

Каждый тест хранится в отдельном JSON файле со следующей структурой:

```json
{
  "id": "test-id",
  "title": "Название теста",
  "icon": "data/{test-id}/images/название_иконки.jpg",
  "description": "Описание теста",
  "questions": [
    {
      "id": 1,
      "text": "Текст вопроса",
      "options": [
        "Вариант ответа 1",
        "Вариант ответа 2",
        "Вариант ответа 3",
        "Вариант ответа 4"
      ],
      "correct": 1
    }
  ]
}
```

Где:
- `id` - уникальный идентификатор теста (латинскими буквами, через дефис)
- `icon` - путь к иконке относительно корня проекта
- `correct` - индекс правильного ответа (начиная с 0)

## Добавление нового теста

1. Создайте папку для нового теста: `data/{test-id}/`
2. Создайте JSON файл `data/{test-id}/{test-id}.json` с вопросами теста (см. формат данных ниже)
3. Создайте папку для иконок: `data/{test-id}/images/`
4. Добавьте иконку теста в папку `data/{test-id}/images/` (формат JPG или PNG)
5. Добавьте информацию о тесте в файл `data/tests-list.json`:
   ```json
   {
     "id": "test-id",
     "title": "Название теста",
     "icon": "data/{test-id}/images/название_иконки.jpg",
     "description": "Описание теста",
     "file": "data/{test-id}/{test-id}.json"
   }
   ```
